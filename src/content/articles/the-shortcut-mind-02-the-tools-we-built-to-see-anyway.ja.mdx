# それでも見るために人類が作ったツール

脳にバイアスがあることを知っても、バイアスは修正されない。そのバイアスを生み出しているのと同じ脳を使って、動機づけられた推論から抜け出すことはできない。だから人間は賢いことをした——脳が見逃すものを捉えるために、脳の外にシステムを構築した。

それが科学の本質だ。単なる事実の集積ではなく、人間の盲点を回避するように設計された一連のルールだ。査読は他の誰かがあなたの推論を確認することを意味する。再現性は、その結果が機能してほしくないと思っている誰かが試みても機能しなければならないことを意味する。事前登録は、データを見る前に自分の予測を書き留めなければならないことを意味する。だから後から予測していたことをひそかに変更できない。このどれもが科学者をより賢くも誠実にもしない。ただ、内部の人々が無意識にそれを欺こうとしているときでさえ、システムを欺きにくくするだけだ。

コンピューターはこれをさらに推し進めた。統計モデル、大規模なデータセット、シミュレーション——これらは人間の直感が最も苦手なタスクを処理する。確率を推定すること、一度に多くの変数を関係の中で保持すること、何千もの事例にわたってパターンを見つけること。脳はこれらすべてが苦手だ。だからこそ、補うためのツールを発明した。

AIはこれと同じことのより興味深いバージョンだ。単に計算するだけでなく——推論するからだ。証拠を見て、議論を構築し、あなたの論理に穴がある場所を指摘できる。そしてあなたと違い、答えに自我を投資していない。守るべき部族もなく、正しくある必要もなく、回避すべき不快感もない。

明らかな問題は、これが自分を欺くための強力なツールにもなるということだ。AIの前は、悪い決断を合理化したければ、限界のある自分の脳でそれをするしかなかった。今は、自分がすでに信じたいことの根拠をAIに求めれば、首尾一貫していて、構造化されていて、反論しにくい何かを生み出してくれる。ツールは中立だ。欲求はそうではない。

だから、AIが人々をより明確に考えるのに役立つかどうかは、ほぼ完全に、それを使う人が実際に明確に考えたいと思っているかどうかにかかっている。それがより難しい問題だとわかる。
