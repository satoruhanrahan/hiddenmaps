# ヒメヒコの回帰としてのジェンダー革命

過去二世紀の最も重要な政治的変容——公的、制度的、聖なる生活における女性の権威の段階的な再主張——は、三極フレームワークを通じて、ヒメヒコ制が数世紀の天的封じ込めの後に地の具現化を要求するものとして読める。

パターンはあらゆるスケールで保たれる：川の民の階層が地の女性的権威を抑圧するとき、聖なる女性は上方に移される——地から排除されながら天で尊重される。抑圧が徹底的であるほど、天の女性はより強く高められる傾向がある。補償的な需要は本物だ。封じ込めがポイントだ。

現代のジェンダー革命はこの操作の逆転だ。女性が地の権威を得るにつれ——政治において、企業において、宗教的制度において、職業において——天の女性への補償的な需要は薄れる。女神は天から降り始める。神学的な決断によってではなく、地の現実が変わっているからだ。

日本の特定のケースは矛盾を最も可視にする。日本は同時に、世界で最も古く最も精巧な女性の聖なる権威の伝統のひとつを持つ——最高神としての天照大御神、ヒメヒコ二重統治制度、女性の聖なる媒介の巫女の伝統——そして先進国の中で最も低い女性の制度的権威の一つを持つ。女性は国会の議席のごく一部しか占めていない。企業のリーダーシップも同様に男性が支配している。天の姫と地の姫の間のギャップは際立っている。

フレームワークはなぜこれが続くかについての特定の読みを示唆する。天照大御神の天的高揚——ほとんど誰も内宮に近づけないほど清純な社に宿る伝統の最も神聖な人物——は女性の地の権威からの排除とは別ではない。それは同じ操作だ。補償は非常に壮大で、天の姫は非常に徹底的に尊重されているので、地の姫の制度的抑圧は明白な矛盾なしに進行できる。文化は天では絶対的に女性を崇拝し、地では体系的に女性を排除する。

このパターンは可逆的だ。女性が最も徹底的に制度的生活に入った社会からの証拠は、地の現実が変わるにつれて補償的な天的インフレーションがある程度萎むことを示唆している。内在的な女性が体系的に排除されなくなると、超越的な女性への需要は薄れる。

三極フレームワークがこの観察に加えるのは特定の予測だ：女性の地の権威の再主張は単なる社会正義の発展ではない。ヒメヒコ制が自らの不均衡を修正しているのだ。統合にはヒメとヒコの両方が真に機能することが必要だ。天においてのみ女性を尊重する文明は、その統治の極のひとつを機能的な形ではなく永続的に儀礼的な形で運営している文明だ。

女神は降りてきている。ゆっくりと、不完全に、巨大な制度的抵抗に逆らって。しかし均衡が常に必要としてきた方向に。
